# bluesky-analysis

Jupyter Lab [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/jobreu/bluesky-analysis/main)

RStudio [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/jobreu/bluesky-analysis/main?urlpath=rstudio)

Demo notebook for exemplary analyses of *Bluesky* data (text & network) collected with the `R` package [`atrrr`](https://jbgruber.github.io/atrrr/)

Created for the masters-level seminar "Erforschung von politischer Online-Kommunikation mit digitalen Methoden" ("Studying political online communication with digital methods") at the University of Duisburg-Essen (winter term 2025/2026).
